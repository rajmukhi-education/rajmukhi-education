# Rajmukhi Education — Final Release

**Version:** 40.0.0  
**Milestones:** V1–V40  
**Status:** Final Release

## Final application scope
- Student learning, course progress and tests
- Admin course/content/enrollment/result management
- Certificate generation, verification, public showcase and print/save as PDF
- Privacy-aware public student profiles and achievements
- Profile, settings, privacy and notification controls
- TOTP 2FA, recovery codes, trusted sessions and security alerts
- Security audit, session management and personal data export
- Admin analytics and security management

## Deployment
1. Install Node.js.
2. Run `npm install` if dependencies are added later.
3. Run `npm start`.
4. Configure the backend using the included template/config files.

## Important
Before public production deployment, set secure production secrets, HTTPS, database backups, email/SMS provider credentials, and review authentication policies.
